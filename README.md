# IT 자산관리 시스템 (Docker 배포형)

한글 UI 기반의 사내 IT 자산관리 도구입니다.

## 핵심 화면 구성
- 대시보드
- 자산리스트(검색)
- 자산추가

## 상태 분류
- 사용중
- 대기
- 폐기필요
- 폐기완료

## 사용 분류
- 주장비
- 대여장비
- 프로젝트장비
- 보조장비
- 기타장비
- 서버장비
- 네트워크장비

## 주요 기능
- JWT 로그인/권한관리(관리자/사용자)
- 자산 등록/검색/수정/삭제
- 자산리스트에서 상태/분류 빠른 수정
- 자산 수정 팝업(세로형 입력 + 항목 설명)
- 사용자(실사용자)와 담당자(관리책임자) 분리 관리
- 자산코드 양식 직접 정의 및 자동 생성
- 변경이력 카드형 보기(작업자/행동/변경 전후)
- 대시보드 상태별/사용분류별/카테고리별 현황

## 빠른 시작
1. 환경 파일 생성 (PowerShell)
```powershell
Copy-Item .env.example .env
```

2. 컨테이너 실행
```bash
docker compose up --build
```

3. 접속
- 웹 앱: `http://localhost:8000/`
- Swagger 문서: `http://localhost:8000/docs`

## 기본 계정
- 아이디: `admin`
- 비밀번호: `.env`의 `ADMIN_PASSWORD`

운영 전 권장:
- `SECRET_KEY`, `ADMIN_PASSWORD` 변경
