# IT 자산관리 시스템 (Docker 배포형)

한글 UI 기반의 사내 IT 자산/라이선스 통합 관리 도구입니다.

- 백엔드: FastAPI + PostgreSQL
- 배포: Docker Compose
- 인증: JWT (관리자/사용자 권한 분리)

## 메뉴 구조

### 1) 대시보드 (통합)
- 하드웨어 + 소프트웨어 통합 현황 요약
- 상태별/사용 분류별/카테고리별 현황 카드
- 각 수치 클릭 시 해당 조건으로 목록 필터 이동
- 대여장비 만료 임박(7일) 현황 제공

### 2) 하드웨어
하위 탭:
- 자산리스트(검색)
- 자산등록
- CSV 업로드
- 폐기완료

주요 기능:
- 자산 등록/수정/조회/삭제
- 자산 수정 팝업(고정 헤더 + 변경이력 가독화)
- 상태 전환 규칙 자동 처리
  - 대기/폐기필요/폐기완료 전환 시 사용자 자동 해제
  - 사용중 전환 시 사용자 지정 유도
- 사용자/담당자 분리 관리
- 사용자 선택: 검색형 드롭다운 + 키보드(방향키/엔터)
- 자산코드 자동 생성 (`{CAT}`, `{YY}`, `{SEQ}` 등)
- `{SEQ}`: 동일 코드 양식 그룹 내 최대값 + 1 규칙
- 폐기완료 자산 분리 관리(폐기 취소/영구 삭제)
- 목록 페이징(50건 단위)
- 하드웨어 자산 리스트 CSV 다운로드(한글 호환)
- 하드웨어 등록 폼: 소프트웨어와 동일한 좌측 제목형 입력 레이아웃

### 3) 소프트웨어
하위 탭:
- 라이선스 등록-수정
- CSV 업로드
- 라이선스 리스트
- 할당 전체 리스트
- 사용자 할당 관리

주요 기능:
- 라이선스 등록/수정
- 라이선스 구분/구독형태/총보유수량/기안자/메모 관리
- 사용자별 할당 관리(할당 시작일/만료일/구매방식 개별 조정)
- 할당 수량/미할당 수량 자동 계산
- 만료 임박/만료 상태 집계
- 라이선스 리스트와 할당 전체 리스트 분리 조회
- 컬럼 정렬, 검색 대상 드롭다운 + 키워드 검색

### 4) 설정
하위 탭:
- 하드웨어 설정
- 소프트웨어 설정
- 사용자
- 관리자
- LDAP

주요 기능:
- 하드웨어 설정: 자산코드 양식, 기본 사용자/담당자, 카테고리 및 `{CAT}` 토큰
- 소프트웨어 설정: 라이선스 구분(추가/수정/삭제)
- 사용자/관리자 탭 분리 운영
- LDAP 연동:
  - 연결 테스트/검색
  - LDAP 검색 결과를 사용자 탭으로 일괄 반영
  - 동기화 스케줄 설정
  - 동기화 시 기존 비활성 사용자 상태 유지
  - Bind 비밀번호는 세션 메모리 사용(로컬 저장 안 함)

## CSV 기능
- 하드웨어 CSV 업로드: 하드웨어 전용 양식
- 소프트웨어 CSV 업로드: 소프트웨어 전용 양식
- 두 영역의 CSV 등록 페이지를 분리 운영

## 빠른 시작
1. 환경 파일 생성
```powershell
Copy-Item .env.example .env
```

2. 컨테이너 빌드/실행
```bash
docker compose up -d --build
```

3. 접속
- 웹 앱: `http://localhost:8000/`
- API 문서(Swagger): `http://localhost:8000/docs`

## 기본 계정
- 아이디: `.env`의 `ADMIN_USERNAME` (기본 `admin`)
- 비밀번호: `.env`의 `ADMIN_PASSWORD` (기본 `ChangeMe123!`)

## 운영 전 권장 사항
- `SECRET_KEY`를 충분히 긴 랜덤 문자열로 변경
- `LDAP_BIND_PASSWORD_KEY` 별도 설정 권장 (LDAP Bind 비밀번호 암호화 저장 키)
- `ADMIN_PASSWORD` 즉시 변경
- DB 계정/비밀번호(`POSTGRES_*`) 운영 환경 기준으로 변경
