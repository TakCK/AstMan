자산 등록

입력 항목 안내

  • 자산명: 자산을 구분하는 대표 이름
  • 카테고리: 장비 종류(설정 탭에서 목록/토큰 관리)
  • 사용 분류: 주장비/대여장비/프로젝트장비 등 운영 목적
  • 대여 시작일자: 대여장비일 때 실제 대여 시작일
  • 대여 만료일자: 대여장비일 때 반납/계약 종료 예정일
  • 상태: 사용중/대기/폐기필요/폐기완료 상태
  • 사용자: 장비를 실제 사용하는 사람(사용중일 때 필수)
  • 담당자: 자산 관리 책임자(운영/구매/정산 담당)
  • 부서: 자산이 소속된 조직 또는 팀
  • 위치: 자산의 현재 물리적 보관 위치
  • 제조사: 벤더/브랜드 정보
  • 모델명: 정확한 모델 식별 정보
  • 시리얼번호: 제조사 고유 식별 번호(중복 불가)
  • 자산코드: 내부 자산 식별 코드(설정 탭 양식 기준 자동생성 가능)
  • 구매처: 구매한 업체 또는 파트너사
  • 구매일: 실제 구매 일자
  • 보증만료일: AS/보증 종료 예정일
  • 구매금액: 내부 기준 자산 취득가
  • 메모: 점검, 특이사항, 운영 참고 정보
  • 상태 변경 규칙: 대기/폐기계열은 사용자 자동 해제, 사용중은 사용자 지정 필요

자산코드 설정

  • {CAT}: 카테고리별 토큰값 (설정 탭의 카테고리 관리에서 지정)
  • {USG}: 사용 분류 약어 (예: 주장비=PRI, 서버장비=SRV)
  • {YYYY}/{YY}: 연도 4자리/2자리
  • {MM}/{DD}: 월/일 2자리
  • {SEQ}: 일자 기준 4자리 자동 증가 번호
  • {SEQ4}: 지정 자리수 증가 번호 (예: {SEQ3}, {SEQ5})
  • {RAND4}: 4자리 랜덤 숫자
